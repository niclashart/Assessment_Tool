
README – Prognose-Tool 

1. Voraussetzungen
-------------------
Installiere die notwendigen Python-Pakete mit:

    pip install -r requirements.txt

Stelle sicher, dass Python 3.x installiert ist und du Zugriff auf eine Konsole oder ein Terminal hast.


2. Prognose für neue Bewerberdaten (Excel-Datei)
--------------------------------------------------
Um für neue Bewerber:innen den Score zu berechnen:

    python prognose_tool_ethisch.py --test testdatei.xlsx
